Full Stack Food Recipe Finder Platform

Designed and developed a comprehensive full-stack recipe discovery platform centered on user personalization, scalable data handling, and intuitive UI. Implemented secure authentication, advanced recipe search and filtering, full CRUD functionality, user-generated content, admin controls, and an integrated food blogging system with commenting. Optimized performance, responsiveness, and UX using server-side rendering and modern design systems to deliver a production-ready application.

Tech Stack: Next.js, Typescript, MongoDB, NextAuth, Cloudinary, Tailwind CSS, Material UI, Kaggle Dataset

To run this project: 
  1. Open backend and frontend folder in terminal and run npm i.
  2. Run npm run dev in both terminals.
  3. Open http://localhost:3000/ and explore my website.
